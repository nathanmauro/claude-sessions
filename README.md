# claude-sessions

Browse, resume, and visualize your Claude Code sessions from three surfaces — terminal CLI, macOS menubar, and a local web dashboard — all backed by a shared SQLite index of `~/.claude/projects/*.jsonl`.

> Work in progress: consolidating [`claude-session-menu`](https://github.com/nathanmauro/claude-session-menu) and [`claude-dash`](https://github.com/nathanmauro/claude-dash) into a single repo.
